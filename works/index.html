PortfolioMario

C#フォームアプリ

周回でどんどん難易度が上がっていく１面構成のスーパーマリオのようなランゲーム。あえてUnityではなくフォームアプリで作っており 55kbと軽量です。開発期間は１週間ほどです。

解かせる数独

JavaScript & Java

数独(ナンバープレイス、ナンプレ)の問題を記入し、JavaScriptによる解析で解答を導き出します。Javaによる問題作成プログラムも掲載しています。空の問題に対しても解答の内の一つを生成します。

ログイン機能

JavaScript

ハンバーガーメニューから表示名（ID）を取得できます。タブを閉じるまで有効（セッションスコープ）です。バックエンド言語も使えますので、サーバー稼働中ずっと有効な会員登録も書くことが可能です。サーブレット、JSP、MySQLを習得しています。

車旅！CarTuber！

JavaScript & PyScript (python)

好きな車に乗り、SNSや講演で稼ぎながら日本全国を旅するオリジナルのシミュレーションゲームです。Pythonでグラフを出力、JavaScriptで変数を計算しています。画像や動画はフリー素材を使っています。
